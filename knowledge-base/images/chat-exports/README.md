# Chat Exports Images Directory

このディレクトリは、故障履歴のチャットエクスポートに含まれる画像ファイルを保存するために使用されます。

## 構造
- 故障履歴のJSONデータから抽出された画像ファイルがここに保存されます
- ファイル名は重複を避けるためにUUIDベースで生成されます
- 元のファイル名はデータベースの `fault_history_images` テーブルに記録されます

## 環境設定
- ローカル開発時: このディレクトリを使用
- 本番環境: 環境変数 `FAULT_HISTORY_IMAGES_DIR` で指定されたディレクトリを使用

## アクセス制御
- 画像ファイルへのアクセスは認証が必要です
- APIエンドポイント `/api/fault-history/images/:filename` 経由でアクセス
